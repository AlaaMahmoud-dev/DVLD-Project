<h1>DVLD Project July/2024-Sep/2024</h1>



<h2>Description</h2>
Driver and License application for managing variant applications such as new licenses, Renew Licenses, replacement lost or damaged liceses and detained licenses.
with wide managing for users and people in system
br />








<h2>Technologies Used</h2>

- <b>.Net framework</b> 
- <b>C# Language</b>
- <b>ADO.net</b>
- <b>Windows Forms</b>
